تعلم أساسيات Git و GitHub والعمل التعاوني

الدرس الأول: أوامر Git الأساسية

في هذا الدرس نتعلم كيفية تهيئة المستودع وحفظ التغييرات.

أهم الأوامر هي:
git init - لتهيئة المستودع
git add - لإضافة الملفات
git commit - لحفظ التغييرات

الدرس الثاني: التعاون (Collaborations)

كيفية العمل مع فريق ودمج التغييرات.

للتعاون نستخدم:
git push - لرفع التغييرات
git pull - لسحب التغييرات
git clone - لنسخ المستودع
